# Pomodoro Paper Workbench

一个纸面式番茄工作法工具，包含活动清单、今日待办、计划外紧急区、专注计时、打断记录、最近三天记录和每日总结。

## 功能

- 设置工作分钟和休息分钟。
- 选定待办后开始番茄钟。
- 今日待办支持计划区和计划外紧急区。
- 待办支持 `□`、`○`、`△` 估算格，支持打叉和删除。
- 支持内部打断 `'`、外部打断 `-`，可添加也可删除。
- 专注模式隐藏主界面，只显示计时器和可收起工具栏。
- 一个工作加休息周期结束后停止，等待手动开始下一个番茄钟。
- 记录按天保存，保留最近三天，并支持每日总结。

## 运行

双击 `serve.cmd`，然后打开命令行里显示的地址。

也可以在项目目录运行：

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

然后打开：

```text
http://localhost:4173/index.html?fresh=20260624-daily-records
```

## 数据保存

数据保存在浏览器的 `localStorage` 中，只在当前浏览器本地保存。上传或分享项目代码不会包含你的个人待办、记录和总结。
